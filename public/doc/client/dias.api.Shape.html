factory: Shape

.api. Shape

Shape() → {Object}

Provides the resource for shapes.

Dependencies:
  • $resource service
Returns:

A new ngResource object

Example
// get all shapes
var shapes = Shape.query(function () {
   console.log(shapes); // [{id: 1, name: "point"}, ...]
});

// get one shape
var shape = Shape.get({id: 1}, function () {
   console.log(shape); // {id: 1, name: "point"}
});

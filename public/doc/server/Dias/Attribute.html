Attribute

class Attribute extends Model

Some models can have one or many of these flexible attributes.

Attributes are imformation on an object that is very seldom assigned (so no extra database column is created for it).

Properties

static array $createRules Validation rules for creating a new attribute
static array $attachRules Validation rules for attaching a new attribute to a model
static array $updateRules Validation rules for updating an attribute of a model
boolean $timestamps Don't maintain timestamps for this model.
protected array $hidden The attributes excluded from the model's JSON form.
